Què és ProPulse?

ProPulse és un projecte d’integració tecnològica: combina maquinari de sensors, xarxes, seguretat, aplicacions web i bases de dades per construir un ecosistema complet de monitoratge esportiu.

Maquinari

Sensors al terreny de joc

Recollida de freqüència cardíaca, accelerometria i esdeveniments (sprints, canvis de direcció, impactes) mitjançant microcontroladors i mòduls sense fils.

Xarxa i seguretat

Comunicació fiable

Enviament de dades xifrades via Wi-Fi/LAN a un servidor central, amb control d’accessos i segmentació de la xarxa per a l’aula de pràctiques.

Web & dades

Quadre de comandament

Interfície web responsiva per visualitzar sèries temporals, resums de sessió i informes bàsics d’evolució del rendiment per esportista.

Mòduls del sistema

L’arquitectura de ProPulse es pot desplegar per fases, ideal per a un projecte de 8 setmanes o per assignatures diferents (maquinari, xarxes, web, BBDD).

01 · Captura

Node de sensors

Codi en microcontrolador (ex. Arduino/ESP32) que llegeix sensors i envia trames en format JSON a la xarxa local.

02 · Servidor

API i base de dades

Servidor Linux amb API (Flask/PHP) i MySQL per emmagatzemar sessions, esportistes i mètriques agregades.

03 · Frontend

Aplicació web

Interfície HTML/CSS/JS que consulta l’API i mostra gràfiques, taules i alertes bàsiques en temps (quasi) real.

Equip ProPulse

El projecte està pensat per treballar en equip, repartint responsabilitats per àrees tècniques i fomentant la documentació i la coordinació.

Maquinari & IoT
Equip de sensors

Web & dades
Equip de programari

Implementació del backend, base de dades i interfície web d’entrenador i esportista, així com informes finals.
